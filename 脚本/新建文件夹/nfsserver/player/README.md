Bash-Games
===========
Linux终端下的小游戏，和小东西，包括 俄罗斯方块，贪吃蛇，屏保时钟，翻译脚本。。。
所有的方向按键都是按照 vim ‘hjkl’ 来的！

数字时钟 (Clock)
================
ClockSaver.sh 利用了 logkeys这个工具(需要root权限), 所以可以达到屏保的目的，无操作时显示数字时钟

aclock.sh 则不需要root 权限, 因为他就仅仅用来显示数字时钟
