shell 中遇到的一些小错误

shell报错：-bash: [: ==: 期待一元表达式 解决方法
问题背景：

test() {     #定义测试参数
        for ((i-0;i<=5;i++))
        do
                if [ $i -eq 3 ]
        then
        $*;   #接受函数外的参数，将会是上四个参数之一
        fi
        echo $i
        done
        echo -e "\E[1;31m i am in func\E[0m;" #循环外的提示
}
        test $*  #这里的“$*” 为函数的传参


如上代码, 执行shell报错：-bash:[: -eq: 期待一元表达式
解决方法：

看见 $i 变量了吗，必须加上引号，因为这里用的是 [ ] 条件修饰符; 所以反过来如果用了 [[]] 条件修饰符就没问题了
